# Boilerplate Site

A tiny starter with an `index.htm`, a stylesheet, and an assets directory.

## Structure

```
boilerplate-site/
├─ index.htm
└─ assets/
   ├─ css/
   │  └─ style.css
   ├─ js/
   │  └─ main.js
   ├─ img/
   └─ fonts/
```

## Usage

Open `index.htm` in your browser or serve the folder with any static server.
